# 4. Fälle: Nominativ & Akkusativ

В немецком языке 4 падежа. На уровне A1 мы учим первые два.

## Nominativ (Именительный падеж)

Отвечает на вопрос **Wer?** (Кто?) или **Was?** (Что?).
Это подлежащее (субъект) в предложении.

*   **Der Mann** schläft.
*   **Das Kind** spielt.

## Akkusativ (Винительный падеж)

Отвечает на вопрос **Wen?** (Кого?) или **Was?** (Что?).
Это прямое дополнение (объект). Используется после многих глаголов: *haben, sehen, kaufen, essen, trinken, lesen...*

**Важно:** В Akkusativ меняется только **мужской род** (der -> den, ein -> einen). Остальные остаются как в Nominativ.

| Род | Nominativ | Akkusativ |
| :--- | :--- | :--- |
| Maskulinum (m) | **der** / ein | **den** / einen |
| Femininum (f) | die / eine | die / eine |
| Neutrum (n) | das / ein | das / ein |
| Plural (pl) | die | die |
