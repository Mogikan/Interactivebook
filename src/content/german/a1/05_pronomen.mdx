# 5. Pronomen (Местоимения)

## Личные местоимения (Personalpronomen)

Мы их уже встречали при спряжении глаголов.

*   ich (я)
*   du (ты)
*   er (он), sie (она), es (оно)
*   wir (мы)
*   ihr (вы - группа людей, друзья)
*   sie (они), Sie (Вы - вежливая форма)

## Притяжательные местоимения (Possessivpronomen)

Отвечают на вопрос **Wessen?** (Чей?).

*   ich -> **mein** (мой)
*   du -> **dein** (твой)
*   er -> **sein** (его)
*   sie -> **ihr** (ее)
*   es -> **sein** (его)
*   wir -> **unser** (наш)
*   ihr -> **euer** (ваш)
*   sie -> **ihr** (их)
*   Sie -> **Ihr** (Ваш)

Окончания меняются как у неопределенного артикля (ein/eine).
*   Das ist mein Vater (m).
*   Das ist mein**e** Mutter (f).
