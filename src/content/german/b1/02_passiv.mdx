# Das Passiv

Im Passiv steht die **Handlung** im Vordergrund, nicht die Person, die sie ausführt (das Subjekt ist oft unwichtig oder unbekannt).

## Bildung

**werden** (konjugiert) + **Partizip II**

| Aktiv | Passiv |
| :--- | :--- |
| Der Mann repariert das Auto. | Das Auto **wird** (vom Mann) **repariert**. |
| Ich koche die Suppe. | Die Suppe **wird gekocht**. |
