# Der Genitiv

Der Genitiv zeigt an, wem etwas gehört ("Wessen?").

## Artikel im Genitiv

| Genus | Nominativ | Genitiv | Endung Nomen |
| :--- | :--- | :--- | :--- |
| Maskulin | der Mann | **des** Mann**es** | -(e)s |
| Neutrum | das Kind | **des** Kind**es** | -(e)s |
| Feminin | die Frau | **der** Frau | - |
| Plural | die Kinder | **der** Kinder | - |

### Übung 2: Endungen ergänzen

Ergänzen Sie die Endung, wenn nötig. Wenn keine Endung nötig ist, lassen Sie das Feld leer oder schreiben Sie "-". (Hier: einfach leer lassen, wenn nichts hinkommt, oder die Endung schreiben).
*Hinweis: Schreiben Sie nur die Endung (z.B. "es" oder "s").*

1. Das Haus des Mann[es] ist groß.
2. Die Farbe des Auto[s] ist rot.
3. Die Schuhe der Frau[] sind neu. (keine Endung)
4. Das Buch des Schüler[s] liegt hier.
