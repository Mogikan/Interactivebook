# Passiv in allen Zeiten

Das Hilfsverb **werden** verändert sich.

| Zeit | Beispiel |
| :--- | :--- |
| Präsens | Es **wird** gemacht. |
| Präteritum | Es **wurde** gemacht. |
| Perfekt | Es **ist** gemacht **worden**. |
| Plusquamperfekt | Es **war** gemacht **worden**. |
| Futur I | Es **wird** gemacht **werden**. |

## Zustandspassiv
Beschreibt einen **Zustand** (Ergebnis), keinen Vorgang.
*   Bildung: **sein** + **Partizip II**.
*   "Die Tür **ist geöffnet**." (Zustand) vs. "Die Tür **wird geöffnet**." (Jemand macht es gerade).
