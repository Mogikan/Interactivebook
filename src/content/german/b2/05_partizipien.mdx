# Partizipien als Adjektive

Wir können Verben wie Adjektive benutzen.

## 1. Partizip I (Gleichzeitigkeit / Aktiv)
Endung: **-d** (+ Adjektivendung)
*   Das Kind lacht. -> Das **lachende** Kind.
*   Der Hund bellt. -> Der **bellende** Hund.

## 2. Partizip II (Vorzeitigkeit / Passiv)
Form: **ge-** (+ Adjektivendung)
*   Das Ei wurde gekocht. -> Das **gekochte** Ei.
*   Das Auto ist repariert. -> Das **reparierte** Auto.
