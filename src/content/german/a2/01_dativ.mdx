# 1. Dativ (Дательный падеж)

Отвечает на вопрос **Wem?** (Кому?) или **Wo?** (Где? - с предлогами).

## Артикли в Dativ

| Род | Nominativ | Akkusativ | Dativ |
| :--- | :--- | :--- | :--- |
| Maskulinum (m) | der / ein | den / einen | **dem** / einem |
| Femininum (f) | die / eine | die / eine | **der** / einer |
| Neutrum (n) | das / ein | das / ein | **dem** / einem |
| Plural (pl) | die | die | **den** + -n |

## Глаголы с Dativ

Некоторые глаголы требуют Dativ:
*   **helfen** (помогать)
*   **danken** (благодарить)
*   **gefallen** (нравиться)
*   **gehören** (принадлежать)
*   **schmecken** (быть вкусным)

### Упражнение 2: Личные местоимения в Dativ

*   ich -> **mir**
*   du -> **dir**
*   er -> **ihm**
*   sie -> **ihr**
*   wir -> **uns**
*   ihr -> **euch**
*   sie/Sie -> **ihnen/Ihnen**
