# 3. Perfekt (Прошедшее время)

Используется в разговорной речи для описания прошлого.

Формула: **haben / sein** (на 2-м месте) + **Partizip II** (в конце).

## Haben или Sein?

*   **sein**: движение (идти, ехать), изменение состояния (просыпаться, умирать) и глаголы *sein, werden, bleiben*.
*   **haben**: все остальные глаголы.

## Построение Partizip II

*   Регулярные (слабые): **ge** + корень + **t** (machen -> **ge**mach**t**)
*   Нерегулярные (сильные): **ge** + корень (часто меняется) + **en** (sehen -> **ge**seh**en**, gehen -> **ge**gang**en**)
