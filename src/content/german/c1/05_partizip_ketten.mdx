# Partizipketten

Im Deutschen können wir sehr viele Informationen **vor** das Nomen packen (als erweitertes Attribut).

*   "Der Mann, der gestern im Park ein Buch las..."
*   -> "Der **gestern im Park ein Buch lesende** Mann..."

## Struktur
Artikel + [Erweiterung] + Partizip + Nomen.

### Übung 2: Relativsatz zu Partizip

Formen Sie um.
<InlineBlanks>
1. Das Problem, das gelöst wurde. -> Das [gelöste] Problem.
2. Die Kinder, die im Garten spielen. -> Die im Garten [spielenden] Kinder.
3. Der Zug, der pünktlich ankommt. -> Der pünktlich [ankommende] Zug.
</InlineBlanks>
